## Kubernetes cluster (Raspberry Pi)

The 4 × Raspberry Pi 4B (8 GB) will be wired up as a **Kubernetes cluster** for
hands-on labbing. Container orchestration is intentionally kept off the single Proxmox
host today — Swarm or k8s there would be overkill — so the Pis will carry that
experiment instead.
